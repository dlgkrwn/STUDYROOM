## print 함수 기본

- 파이썬에서 화면에 출력할 때는 print 함수 사용
- print 함수 안에 출력할 숫자를 작성하면 해당 숫자가, 출력할 문자열을 작성하면 해당 문자열이 화면에 출력됨
- 문자열은 작은따옴표, 큰따옴표로 감싸서 표현해야 함

```python
print(3.14)
>>> 3.14
print("Hello")
>>> Hello
```

## 값을 2개 이상 출력

- 숫자형과 문자열 데이터를 두 개 이상 출력할 때는 콤마( , )를 사용하면 자동으로 띄어쓰기가 적용되어 출력됨

```python
print("Hello", 2.7)
>>> Hello 2.7
print("Hello", "Python")
>>> Hello Python
```

## 줄바꿈 출력 여부

- print 함수가 종료되면 줄바꿈을 함
- 줄바꿈을 하지 않으려면 print 함수에서 end 옵션을 사용하여 설정 가능
- end 옵션의 기본값은 end=”\n”
- 줄바꿈을 제거하고 싶을 때는 end=””와 같이 빈 문자열을 지정
