    

## 3. 숫자 10과 20을 변수 x와 y에 저장하고 , 두 변수의 값을 서로 교환한 후에 출력하는 코드를 작성해보자

- 실행결과 - x : 20, y : 10
    
    ```python
    x, y = 10, 20
    x, y = y, x
    print("x:", x)
    print("y:", y)
    ```
    

## 4. print, input 함수를 둘 다 사용하여 사용자로부터 문자열을 입력받는 코드를 작성해보자

- 입력 - Hello, 실행결과 - 입력: ‘Hello’
    
    ```python
    print("입력:")
    input()
    ```
    

## 5.두 개의 숫자를 입력받아 두 숫자를 출력하는 코드를 작성해보자.

- 입력 - 입력1 : 8, 입력2 : 5, 실행결과 - 8, 5
    
    ```python
    x = int(input("입력1:"))
    y = int(input("입력2:"))
    print(x)
    print(y)
    ```
