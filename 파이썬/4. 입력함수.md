## input 함수 기본

- 사용자로부터 문자열 형태의 데이터를 입력받아 반환하는 함수

input(prompt)

- prompt는 사용자에게 입력을 요청할 때 표시할 메시지를 문자열로 지정

```python
x = input()
print(x)
>>> [20] # 살행하면 [] 칸이 나옴 []안에 20을 넣으면
>>> 20

y = input("입력해주세요:")
print(y)
>>> 입력해주세요:[40] # []칸에 40을 넣으면
>>> 40
```

## 정수형 입력

- int 함수는 문자열을 정수로 변환하는 함수

int(input(prompt))

```python
x = int(input("정수를 입력해주세요:")
print(x)
>>> 정수를 입력해주세요: [20]
>>> 20
```
